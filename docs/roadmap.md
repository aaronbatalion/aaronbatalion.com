# Roadmap

## Goal: Re-imagine aaronbatalion.com Design

The objective is to transform the current simple list of links into a visually engaging, modern personal hub that aggregates content from various platforms (Medium, X, Threads) with a premium aesthetic.

## Milestones

- [ ] **Phase 1: Concept & Foundation**
    - [x] Update Branding Assets (Headshot, Favicon).
    - [x] Define design direction (Modern/Premium).
    - [x] Set up project structure for new assets (icons/Components).
- [ ] **Phase 2: Core Components**
    - Create `LinkCard` component with favicon support.
    - Implement data fetching/hardcoding for X/Threads links.
- [ ] **Phase 3: Visual Polish**
    - Apply advanced CSS (glassmorphism, animations, typography).
    - Ensure responsive design.
- [ ] **Phase 4: Launch**
    - Final review and deployment.
